# Partner Data Sheet Website

This website allows users to view and search a data sheet of partner organizations.

## Features

* View a table of partner organizations, including their name, email, organization type, and what they provide.
* Search the table by any of the columns.
* Sort the table by any of the columns.

## How to Use

To use the website, simply navigate to the homepage and click on the "View" button. This will take you to a page that displays the table of partner organizations.

To search the table, enter a search term in the search bar at the top of the page. The table will be filtered to only show the rows that contain the search term.

To sort the table, click on the header of the column that you want to sort by. The table will be sorted in ascending order by default. To sort in descending order, click on the header of the column again.
